grid-auto-row: The grid-auto-row property in CSS Grid is used to set the size of implicitly created rows in a grid. These are rows that are generated automatically when grid items are placed in the grid, and the number of rows exceeds the explicitly defined rows in the grid template.

grid-auto-column: The grid-auto-column property in CSS Grid is used to set the size of implicitly created columns in a grid. Similar to grid-auto-row, these columns are generated automatically when grid items are placed in the grid, and the number of columns exceeds the explicitly defined columns in the grid template.

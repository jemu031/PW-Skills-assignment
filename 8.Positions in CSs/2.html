Absolute Positioning Relative Positioning
In CSS, absolute positioning places an element relative to its parent's position. Relative positioning places an element relative to its current position.
Absolute positioning Places an element relative to the first parent element with a position other than static. Relative positioning Places an element relative to its default position on the page.
If no such element is found, the containing block is HTML. Absolute positioning changes the layout around the element. The element still takes up space in the page's normal flow, but it can be shifted in any direction.

Here is a step by step process of contributing to Open-Source project:

  1. Fork the Repository:On the project's GitHub page, click "Fork" to create a personal copy.
  2. Clone the Repository:Clone the forked repository to your local machine using `git clone`.
  3. Create a Branch:Create a new branch for your changes using `git checkout -b feature-branch`.
  4. Make Changes:Make necessary code modifications and additions.
  5. Commit Changes:Stage and commit changes using `git add` and `git commit`.
  6. Push to Your Fork:Push changes to your forked repository with `git push origin feature-branch`.
  7. Create a Pull Request: On GitHub, submit a pull request from your branch to the original repository.
  8. Discuss and Merge:Collaborate with maintainers through comments and, upon approval, have your changes merged into the main project.
